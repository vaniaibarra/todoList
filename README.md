# To-Do List

Aplicación full stack para gestionar tareas con autenticación JWT.

## Tecnologías
- React
- Node.js
- Express
- PostgreSQL
- Supabase
- Render
- Vercel

## Demo

https://todo-list-teal-chi-97.vercel.app/login

## Funcionalidades

- Registro e inicio de sesión
- CRUD de tareas
- Autenticación JWT
- Protección de rutas
